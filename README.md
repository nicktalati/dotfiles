# NT's Dotfiles

This more or less contains my entire Arch Linux config.

## Stack

**OS:** Arch
**WM:** Sway
**Shell:** Zsh
**Secrets:** Cryptomator
**Management:** GNU Stow

## Bootstrapping a fresh Arch install

```bash
sudo pacman -S git
git clone https://github.com/nicktalati/dotfiles.git ~/dotfiles
cd ~/dotfiles
./install_arch.sh
```
